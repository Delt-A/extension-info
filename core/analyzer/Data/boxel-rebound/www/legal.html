User Submitted content

All user submitted content is anonymous, and subject to deletion at any time at our discretion. User submitted content includes "custom levels" that can be shared with other Boxel Rebound users. Users can anonymously report any innappropriate data within the extension. By uploading "custom levels", you agree that all user submitted data belongs to Doppler Creative. User submitted content is transmitted over a web-based secured protocol ("https://"). No web data can ever be 100% secured, but we do our best to prevent data manipulation. User submitted data is stored and accessed with an authorized SSL certificate.
